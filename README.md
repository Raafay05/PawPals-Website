# PawPals - Pet Social Media Website

# How to Run
1. Download [XAMPP Control Panel](https://www.apachefriends.org/download.html) if you don't already have it installed.
2. Download the source code from this repository as a zipped file.
3. Open the zipped file and copy the folder that is inside of it.
4. Open File Explorer and follow the following path `C: > xampp > htdocs`.
5. Inside of the htdocs folder, paste the folder that you copied with the source code here.
6. Using XAMPP control panel, run **Apache** and **MySQL**, click the **Admin** button and it should show you the PawPals database.
7. In your web browser, copy and paste the following link to open the PawPals Website: ```http://localhost/PawPals-Website-Main/registration.html```.
8. The browser should now display the website.
9. Register for an account. **Note: Be sure to keep track of the password as it will be needed to login to the website.**
10. You should now be logged in to your account. Go ahead and use the app and get to know its features and learn about the pet community!
