# PawPals - Pet Social Media Website

Developed a full-stack social media platform for pet owners using JavaScript, PHP, MySQL, HTML, CSS, Google Gemini API, and Leaflet API.

Designed and implemented an intuitive and user-friendly interface featuring user authentication, homepage, VetAI chatbot, pet profile, and settings, enhancing user engagement and navigation.

Integrated VetAI, an AI-powered chatbot utilizing the Google Gemini API to provide pet care recommendations and answer pet related questions in under 8 seconds.

Incorporated the Leaflet Map API to enable geotagged posts, allowing users to discover pet-friendly locations, share experiences, and connect with nearby pet owners.

Developed a relational MySQL database to securely manage user accounts and pet profiles.

# How to Run
1. Download [XAMPP Control Panel](https://www.apachefriends.org/download.html) if you don't already have it installed.
2. Download the source code from this repository as a zipped file.
3. Open the zipped file and copy the folder that is inside of it.
4. Open File Explorer and follow the following path `C: > xampp > htdocs`.
5. Inside of the htdocs folder, paste the folder that you copied with the source code here.
6. Using XAMPP control panel, run **Apache** and **MySQL**, click the **Admin** button and it should show you the PawPals database.
7. In your web browser, copy and paste the following link to open the PawPals Website: ```http://localhost/PawPals-Website-Main/registration.html```.
8. The browser should now display the website.
10. Register for an account. **Note: Be sure to keep track of the password as it will be needed to login to the website.**
11. You should now be logged in to your account. Go ahead and use the app and get to know its features and learn about the pet community!
